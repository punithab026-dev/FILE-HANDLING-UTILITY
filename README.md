*This project is a simple File Handling Utility developed in Java as part of an internship task under CODTECH. It demonstrates basic file operations such as creating a file, writing data into it, reading the stored content, and modifying the file by appending new information.

The application uses standard Java classes like File, FileWriter, and Scanner to perform these operations. It provides a console-based interface where users can enter a file name and content, making it easy to understand how file handling works in Java.

This project helps in building a strong foundation in managing files and handling data efficiently in real-world Java applications.
